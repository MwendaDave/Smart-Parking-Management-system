# Smart Parking Management System

A web-based Smart Parking Management System designed to help drivers locate available parking spaces efficiently while allowing administrators to manage parking slots and reservations.

## 📌 Project Overview
The Smart Parking Management System is designed to reduce time spent searching for parking by providing a centralized system for managing parking spaces. The system allows users to view available parking slots, reserve them, and manage bookings through an intuitive interface.

This project was developed as part of the **CMT 311 course project**.

---

## 🎯 Objectives
- Provide a platform for drivers to find available parking spaces.
- Enable reservation of parking slots in advance.
- Allow administrators to manage parking spaces.
- Improve parking efficiency and reduce congestion.

---

## 🚀 Features
- User Registration and Login
- Parking Slot Availability Display
- Parking Slot Reservation
- Admin Dashboard for Managing Slots
- Booking History
- Real-time parking updates
